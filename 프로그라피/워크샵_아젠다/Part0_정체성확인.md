# Part 0. 프로그라피 정체성 확인

> **진행 방식**: 서면 공유 → 비동기 소통 → 문의 많으면 1/8 회의에서 동기 소통

---

## 배경

10월, 11월 워크숍에서 일부 결정된 사항들입니다.
신규 멤버들의 의견을 반영하고 전체 얼라인을 위해 공유합니다.

---

## 0-1. 프로그라피가 걸어온 길

**과거:**
- 사이드프로젝트 환경을 돕는 IT 동아리
- "강제적인 환경" 제공

**현재:**
- 우리가 잘하는 것: 사이드프로젝트
- 한 단계 더: 사이드프로젝트 → 수익화로 키워드 전환

**앞으로:**
- 방향성: IT 동아리를 넘어 커뮤니티로 변화
- 커뮤니티 활성화 방안으로 6개월 유료 프로그램 제공
- 수익화의 의미: 1원이라도 벌어보자

---

## 0-2. 오늘 우리의 정체성

**우리는:**
- 수익화 커뮤니티 (O)
- 사이드프로젝트로 1원이라도 버는 경험 제공 (O)
- 그 과정을 함께하는 사람들의 모임 (O)

**우리는 아님:**
- VC 투자 받는 스타트업 만드는 곳 (X)
- 창업 교육 기관 (X)
- 취업 포트폴리오 쌓는 부트캠프 (X)

**구조:**
```
➊ [커뮤니티] ← 핵심
➋ [6개월 유료 프로그램] ← 커뮤니티 활성화 수단
➌ [수익화 = 1원이라도] ← 프로그램 목표
```

---

## 0-3. 이미 결정된 것들

- [ ] 우리는 커뮤니티다
- [ ] 6개월 프로그램은 커뮤니티를 만드는 수단
- [ ] 프로그램이 끝나도 커뮤니티는 남아야 함
- [ ] 11기는 진행한다 (2026년 상반기)
- [ ] 수익화 = 1원이라도 벌어보기
