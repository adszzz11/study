# Part 4. 커뮤니티

| 항목 | 내용 |
|------|------|
| 진행 방식 | 1/11 워크숍 (20분) - 시간 부족 시 후속 회의 |
| 목표 | 프로그램 종료 후에도 남는 커뮤니티 만들기 |

---

## 현재 문제

- 참가자 돈의 가치 = 멘토링만
- 멘토 1명 빠지면 무너짐
- 프로그램 끝나면 커뮤니티 사라짐

---

## 활성화 방안 옵션

| 주체 | 방안 | 부담 | 효과 |
|------|------|------|------|
| 멘토 | 월 1회 강연/팟캐스트 | 중 | 콘텐츠 축적 |
| 멘토 | 수익화 경험 공유회 | 낮음 | 동기부여 |
| 참가자 | 팀 간 교류회 | 시스템만 | 커뮤니티 형성 |
| 참가자 | 멘티 세미나 | 낮음 | 참여 유도 |
| 운영진 | 오프라인 만남 월 1회 | 중 | 결속력 |
| 운영진 | 리캡 콘텐츠 | 낮음 | 기록 축적 |

---

## 종료 후 경로 옵션

| 옵션 | 설명 |
|------|------|
| 정규 멤버 전환 | 11기 종료 = 정규 커뮤니티 멤버 |
| 멘토 성장 경로 | 11기 참가자 → 12기 멘토 |
| 수익화 지속 환경 | 프로그램 끝나도 계속 시도 가능 |

---

## 결정할 것

- 11기에 실행할 방안 2-3개
- 각 방안 담당자
- 종료 후 경로 1-2개
