# Part 0. 정체성 확인

> 서면 공유 (1/6) → 비동기 의견 (마감 1/7)

## 프로그라피 정체성

**우리는:**
- 수익화 커뮤니티
- 사이드프로젝트로 1원이라도 버는 경험 제공
- 그 과정을 함께하는 사람들의 모임

**우리는 아님:**
- VC 투자 받는 스타트업 만드는 곳
- 창업 교육 기관
- 취업 포트폴리오 쌓는 부트캠프

## 우리의 약속 (CVP)

> 프로그라피는 0부터 1, 매출로 이어지는 비즈니스 사이클을 실제로 경험할 수 있는 환경을 제공한다.

## 톤앤매너

**DO**
- 숫자로 말한다
- 사실과 의견을 구분한다
- 일단 실행한다
- 기준을 명확히 한다

**DON'T**
- 가정법을 쓰지 않는다
- 결과 없이 위로만 하지 않는다
- "나중에"로 미루지 않는다
- 다른 팀과 비교하지 않는다

## 확인 사항

- 위 정체성에 동의하는가
- 이견이 있으면 1/7까지 비동기로 의견 남기기

# Part 1. 목표 설정

> 서면 공유 (1/6) → 비동기 의견 (마감 1/7)

## 성공의 정의

| 옵션 | 내용 | 장단점 |
|------|------|--------|
| A | 프로그램 완주 | 측정 쉬움 / 커뮤니티 안 남을 수 있음 |
| B | 수익화 달성 | 정체성 부합 / 통제 어려움 |
| C | 커뮤니티 잔류 | 커뮤니티 핵심 / 정의 모호 |
| **D** | **복합** | 균형 / 측정 복잡 |

**권장안**: 옵션 D (완주 + 수익화 시도 + 커뮤니티 잔류)

## 최소 성공 기준 (초안)

| 구분 | 기준 |
|------|------|
| Must | 10개 팀 중 5개 팀 완주 |
| Must | 30명 중 15명 커뮤니티 잔류 |
| Must | 전 팀 매출 시도 (금액 무관) |
| Nice | 3개 팀 실제 매출 발생 |

## 솔루션

**지금 해야 할 것**
- 0→1→매출 경험 있는 멘토/연사
- 서비스 만들 수 있는 수준의 동료만 선발
- 매출 경험 가이드북 제공
- 자유 팀 구성 + 셀프 매칭 가이드

**하지 않을 것**
- 스타트업/투자자 연결
- 다작 프로젝트 (1개 완주에 집중)
- 방대한 자료

## 확인 사항

- 성공 정의 옵션 D에 동의하는가
- Must Have 기준의 숫자가 적절한가

# Part 2. 역할 정의

> 1/8 전체회의 (디스코드)

## 목표

역할과 권한 명문화

## 전제

- 멘토 = 운영진 = 커뮤니티 빌더
- 멘토링은 운영진 업무 중 하나

## 역할/권한 초안

| 도메인 | 역할 | 권한 | 담당 |
|--------|------|------|------|
| 프로그램 운영 | 일정 관리, 공지, 출석 | 일정 변경, 공지 발송 | 오퍼레이터 |
| 멘토링 | 정기 세션, 피드백 | 방향 제안, 경고 발의 | 멘토 |
| 선발 | 서류/면접 심사 | 합격/불합격 결정 | 멘토+오퍼레이터 |
| 퇴출 | 불성실 모니터링 | 경고 발송, 퇴출 결정 참여 | 전체 운영진 |
| 커뮤니티 | 교류 촉진, 이벤트 | 이벤트 기획/실행 | 오퍼레이터 |
| 대외/홍보 | 모집 홍보 | 공식 채널 포스팅 | 지정 담당 |
| 재정 | 예산 관리 | 지출 승인 (한도 내) | 총괄 |

## 권한 레벨

| 레벨 | 설명 | 예시 |
|------|------|------|
| 실행 | 승인 없이 바로 실행 | 멘토링 세션, 팀 피드백 |
| 제안 | 제안 후 합의 필요 | 퇴출 발의, 일정 변경 |
| 승인 | 총괄/전체 회의 승인 필요 | 예산 초과, 규칙 변경 |

## 멘토 현실

- 월 2회, 팀당 주 2시간 이내
- 낮 시간 어려움, 늦은 퇴근 잦음
- 오프라인은 한 달 전 공지 필요

## 결정할 것

- "멘토 = 운영진" 동의 확인
- 역할/권한 리스트 확정
- 권한 레벨 확정

# Part 4. 커뮤니티

> 1/11 워크숍 (20분) - 시간 부족 시 후속 회의

## 현재 문제

- 멘토 1명 빠지면 무너짐
- 프로그램 끝나면 커뮤니티 사라짐

## 활성화 방안 옵션

| 주체 | 방안 | 부담 | 효과 |
|------|------|------|------|
| 멘토 | 월 1회 강연/팟캐스트 | 중 | 콘텐츠 축적 |
| 멘토 | 수익화 경험 공유회 | 낮음 | 동기부여 |
| 참가자 | 팀 간 교류회 | 시스템만 | 커뮤니티 형성 |
| 참가자 | 멘티 세미나 | 낮음 | 참여 유도 |
| 운영진 | 오프라인 만남 월 1회 | 중 | 결속력 |
| 운영진 | 리캡 콘텐츠 | 낮음 | 기록 축적 |

## 종료 후 경로 옵션

| 옵션 | 설명 |
|------|------|
| 정규 멤버 전환 | 11기 종료 = 정규 커뮤니티 멤버 |
| 멘토 성장 경로 | 11기 참가자 → 12기 멘토 |
| 수익화 지속 환경 | 프로그램 끝나도 계속 시도 가능 |

## 결정할 것

- 11기에 실행할 방안 2-3개
- 각 방안 담당자
- 종료 후 경로 1-2개
