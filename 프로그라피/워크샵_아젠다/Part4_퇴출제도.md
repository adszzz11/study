## 4-1. "불성실"의 정의 (15분)

**예시:**
| 행동 | 불성실 여부 | 비고 |

| 주간 공유 2회 연속 누락 | ? | |
| 팀 활동 무단 불참 | ? | |
| 멘토링 세션 불참 | ? | |
| 기타: ___ | | |

## 4-2. 판단 주체와 절차 (15분)

**질문:**
- 누가 판단하는가? (운영진 전원? 특정 인원? 멘토?)
- 경고는 몇 번?
- 어떤 절차로 퇴출?

**예시 프로세스:**
```
1차 경고 (구두) → 2차 경고 (서면) → 3차 퇴출 통보
```

---

## 4-3. 엣지 케이스 논의 (15분)

**질문**: 이런 경우는 어떻게?
- 개인 사정 (건강, 가족 등)으로 인한 불참
- 팀 내 갈등으로 인한 활동 중단
- 팀 전체가 흐지부지 되는 경우
