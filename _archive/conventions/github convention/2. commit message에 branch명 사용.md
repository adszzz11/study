### 💡 왜 브랜치명을 커밋 메시지에 포함할까?

| 목적         | 설명                                      |
| ---------- | --------------------------------------- |
| **추적성 강화** | 커밋이 어떤 기능이나 작업 브랜치에서 유래했는지 바로 확인 가능     |
| **이슈 연결**  | 브랜치명이 Jira 티켓이나 이슈 ID와 연동되면 추적이 쉬움      |
| **리뷰 효율화** | PR 리뷰 시 커밋이 어떤 목적의 브랜치에서 나왔는지 명확히 파악 가능 |

---

### 🧱 기본 형식 예시
`✨ [feat/login] 로그인 화면 구성`
`🐛 [fix/signup-validation] 회원가입 이메일 유효성 검사 수정`
`📝 [docs/readme] README 문서 업데이트`
