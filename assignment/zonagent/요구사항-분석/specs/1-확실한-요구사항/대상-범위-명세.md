## 📄 문서 타입 (Document Types)

### 1. Meeting Minutes (회의록)
| 항목 | 세부사항 |
|------|----------|
| **포맷** | PDF 또는 HTML |
| **설명** | 회의 진행 내용 기록 |
| **중요도** | 🔴 HIGH |
| **구현 우선순위** | 1순위 (가장 먼저 구현) |

**특징:**
- 회의 결과 및 의사결정 내용 포함
- 보통 회의 후 며칠~몇 주 후 게시
- 공식 문서로 법적 효력 있음

**예상 메타데이터:**
- 회의 날짜
- 회의 타입 (정기/임시)
- 문서 게시일
- 승인 여부

---

### 2. Meeting Agendas (회의 안건)
| 항목 | 세부사항 |
|------|----------|
| **포맷** | PDF 또는 HTML |
| **설명** | 회의 예정 안건 목록 |
| **중요도** | 🟡 MEDIUM |
| **구현 우선순위** | 2순위 |

**특징:**
- 회의 전에 게시 (보통 며칠 전)
- Minutes보다 먼저 나옴
- 간략한 안건 리스트

**예상 메타데이터:**
- 회의 예정일
- 회의 시간/장소
- 안건 항목 수

---

### 3. Agenda Packets (안건 패킷)
| 항목 | 세부사항 |
|------|----------|
| **포맷** | PDF (대용량) |
| **설명** | 안건 관련 모든 문서를 묶은 패키지 |
| **중요도** | 🟡 MEDIUM |
| **구현 우선순위** | 3순위 |

**특징:**
- 여러 안건의 상세 자료 포함
- 파일 크기가 큼 (수십~수백 MB)
- 안건과 함께 또는 별도로 게시

**기술적 고려사항:**
- 다운로드 시간 길 수 있음
- 스토리지 용량 고려 필요
- 네트워크 타임아웃 처리

---

### 4. Video Recording Links (비디오 녹화 링크)
| 항목 | 세부사항 |
|------|----------|
| **포맷** | 링크 (Granicus, YouTube, Vimeo) |
| **설명** | 회의 영상 녹화본 |
| **중요도** | 🟢 LOW |
| **구현 우선순위** | 4순위 (선택적) |

**플랫폼:**
- **Granicus**: 정부 전용 비디오 플랫폼
- **YouTube**: 공개 채널
- **Vimeo**: 대안 플랫폼

**수집 범위:**
- 비디오 파일 자체는 다운로드 X
- 링크 URL만 메타데이터로 저장
- 플랫폼 정보 및 비디오 ID 추출

**예상 메타데이터:**
- 비디오 URL
- 플랫폼 (Granicus/YouTube/Vimeo)
- 업로드 날짜
- 길이 (가능한 경우)

## 📊 우선순위 매트릭스

### 문서 타입별 우선순위

| 순위 | 문서 타입 | 이유 | MVP 포함 |
|------|-----------|------|----------|
| 1 | Minutes | 가장 중요한 공식 문서 | ✅ |
| 2 | Agendas | 회의 계획 파악 | ⚠️ (Phase 2) |
| 3 | Packets | 상세 자료 (대용량) | ⚠️ (Phase 2) |
| 4 | Videos | 참고 자료 (링크만) | ❌ (선택적) |

### 지자체별 우선순위 (사이트 분석 후 결정)

**결정 기준:**
1. 웹사이트 구조의 단순성
2. 문서 접근 용이성 (로그인 불필요)
3. 최신 기술 사용 여부 (정적 vs 동적)
4. 문서 업데이트 빈도

**분석 필요:**
- 각 지자체 웹사이트 수동 탐색
- 문서 페이지 URL 패턴 파악
- 스크래핑 난이도 평가

- ✅ **2개 지자체**

### Ideal (Full Implementation)
- ✅ **4개 지자체 모두**
- ✅ **4개 문서 타입 모두**
- ✅ **1-2년** 데이터

## 📝 데이터 수집 체크리스트

### 각 문서 수집 시 필수 정보

```python
{
    "jurisdiction": str,        # "cherokee_county", "holly_springs", ...
    "doc_type": str,            # "minutes", "agenda", "packet", "video"
    "meeting_date": date,       # 회의 날짜
    "title": str,               # 문서 제목
    "url": str,                 # 원본 URL
    "file_path": str,           # 로컬 저장 경로 (비디오 제외)
    "file_format": str,         # "pdf", "html", "link"
    "file_size": int,           # 바이트 단위 (해당 시)
    "downloaded_at": datetime,  # 다운로드 시각
    "checksum": str,            # MD5/SHA256 (중복 방지)
}
```
