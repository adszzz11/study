# 마케팅 치트시트

## 기본 흐름

```text
Market -> Customer -> Value Proposition -> STP -> 4P -> Campaign -> Measurement
```

## 핵심 프레임워크

| Framework | 구성 | 한 줄 용도 |
|---|---|---|
| `3C` | Customer, Company, Competitor | 시장 상황을 빠르게 진단 |
| `SWOT` | Strength, Weakness, Opportunity, Threat | 전략 이슈 정리 |
| `STP` | Segmentation, Targeting, Positioning | 누구에게 어떤 위치로 팔지 결정 |
| `4P` | Product, Price, Place, Promotion | 실행 mix 설계 |
| Journey | Awareness, Consideration, Purchase, Retention | 고객 접점 설계 |
| KPI Tree | Goal -> KPI -> Metric | 측정 구조 설계 |

## Positioning Statement

```text
For [target segment],
[brand/product] is the [category]
that provides [primary benefit]
because [reason to believe].
```

## 소비자 행동 키워드

| Term | 의미 |
|---|---|
| Need recognition | 문제나 욕구를 인식하는 순간 |
| Information search | 대안과 정보를 찾는 단계 |
| Evaluation | 대안을 비교하는 단계 |
| Social proof | 다른 사람의 선택을 신뢰 신호로 쓰는 경향 |
| Loss aversion | 이익보다 손실을 더 크게 느끼는 경향 |
| Anchoring | 처음 본 숫자나 정보가 판단 기준이 되는 현상 |

## 디지털 마케팅 지표

| Metric | 계산 |
|---|---|
| CTR | clicks / impressions |
| CPC | cost / clicks |
| CVR | conversions / visits |
| CAC | marketing cost / new customers |
| LTV | average revenue per customer * gross margin * retention period |
| ROAS | revenue / ad spend |
| Retention | retained users / cohort users |

## 캠페인 설계 체크리스트

- [ ] Objective가 business goal과 연결되는가?
- [ ] Target segment가 충분히 구체적인가?
- [ ] Customer insight가 실제 조사나 데이터에서 나왔는가?
- [ ] Main message가 하나로 정리되는가?
- [ ] Channel별 역할이 다른가?
- [ ] Primary KPI와 guardrail metric이 정해졌는가?
- [ ] UTM과 tracking plan이 준비됐는가?
