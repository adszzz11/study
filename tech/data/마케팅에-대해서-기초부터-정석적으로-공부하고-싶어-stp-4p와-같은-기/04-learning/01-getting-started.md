# 시작하기: 0-3개월차

> [[../README|목차로 돌아가기]] | [[02-deep-dive|다음: 심화]]

## 목표

0-3개월차의 목표는 채널 운영보다 먼저 `시장`, `고객`, `가치제안`, `포지셔닝`을 말로 설명할 수 있게 되는 것이다.

```text
좋은 첫 목표:
"이 브랜드는 누구에게, 어떤 문제를, 어떤 이유로, 어떤 방식으로 팔고 있는가?"
```

---

## 0-1개월차: 마케팅 원론

### 학습 개념

- AMA marketing definition
- `Market`, `Customer value`, `Exchange`, `Brand`
- `3C`: Customer, Company, Competitor
- `SWOT`, `PESTEL`, category analysis
- `STP`: Segmentation, Targeting, Positioning
- `4P`: Product, Price, Place, Promotion

### 실습 1: 브랜드 3개 분석

| 항목 | 질문 |
|---|---|
| Target segment | 이 브랜드의 핵심 고객은 누구인가? |
| Need/Pain | 고객은 어떤 문제나 욕구를 갖는가? |
| Positioning | 고객 머릿속에서 어떤 위치를 차지하는가? |
| Product | 제품/서비스의 핵심 benefit은 무엇인가? |
| Price | 가격은 premium, parity, low-cost 중 어디인가? |
| Place | 어디서 발견되고 구매되는가? |
| Promotion | 어떤 메시지와 채널을 쓰는가? |

```markdown
## Brand Teardown

- Brand:
- Category:
- Target segment:
- Main pain point:
- Positioning statement:
- 4P:
  - Product:
  - Price:
  - Place:
  - Promotion:
- 한 줄 평가:
```

### 추천 브랜드

- Nike: brand identity와 community
- Apple: premium positioning과 ecosystem
- Toss: simplicity, trust, product-led growth
- 무신사: vertical commerce와 youth culture

## 2-3개월차: 소비자 행동

### 학습 개념

- 구매 의사결정 5단계
  - need recognition
  - information search
  - evaluation
  - purchase
  - post-purchase behavior
- motivation, perception, attitude, memory
- social proof, loss aversion, anchoring, default effect
- customer journey와 touchpoint

### 실습 2: 리뷰 100개 분류

제품 하나를 고르고 리뷰 100개를 수집해 아래 표로 분류한다.

| 분류 | 설명 | 예시 |
|---|---|---|
| Pain point | 구매 전/사용 중 불편 | "설치가 어렵다" |
| Benefit | 고객이 얻은 이익 | "시간이 절약된다" |
| Barrier | 구매를 막는 요인 | "가격이 부담된다" |
| Trigger | 구매를 시작하게 한 계기 | "친구 추천" |
| Language | 고객이 실제 쓰는 표현 | "가성비", "안심" |

```csv
review_id,quote,pain_point,benefit,barrier,trigger,emotion
1,"배송이 빨라서 급할 때 좋았다",,빠른 배송,,급한 필요,positive
2,"가격이 조금 비싸서 고민했다",,,가격 부담,,negative
```

### 실습 3: Customer Journey Map

| 단계 | 고객 질문 | Touchpoint | 감정 | 개선 기회 |
|---|---|---|---|---|
| Awareness | 이 문제를 어떻게 알게 되는가? | Search, SNS, 지인추천 | 호기심 | 문제 정의 콘텐츠 |
| Consideration | 어떤 대안을 비교하는가? | Review, 비교글, 상세페이지 | 불안 | 비교표, FAQ |
| Purchase | 왜 지금 사는가? | Landing page, checkout | 확신/망설임 | 보증, 할인, 후기 |
| Retention | 다시 쓰는 이유는? | Email, app push, community | 만족/귀찮음 | onboarding, CRM |

## 체크리스트

- [ ] `STP`와 `4P`를 구분해서 설명할 수 있다.
- [ ] 브랜드 3개를 같은 템플릿으로 비교했다.
- [ ] 리뷰 100개를 pain point, benefit, barrier로 나눴다.
- [ ] 고객 여정의 주요 touchpoint를 그렸다.
- [ ] positioning statement를 3개 이상 써봤다.
