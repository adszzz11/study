## Claude Code v2.1.149 (2026-05-22) ⭐

> 출처: https://code.claude.com/docs/en/changelog | https://github.com/anthropics/claude-code/releases/tag/v2.1.149

### 신규 기능

#### `/usage` 카테고리별 상세 내역
- 사용량이 **skills / subagents / plugins / MCP 서버별** 세분화되어 표시
- 어떤 기능이 한도를 소모하는지 명확하게 파악 가능

```bash
/usage
# → Skills: 30% | Subagents: 45% | Plugins: 15% | MCP: 10%
```

#### `/diff` 키보드 스크롤 지원
- 상세보기 패널에서 키보드로 스크롤 가능
- 지원 키: 화살표, `j`/`k`, `PgUp`/`PgDn`, `Space`, `Home`/`End`

#### GFM 태스크 리스트 체크박스 렌더링
- `- [ ] 할 일` / `- [x] 완료` 문법이 이제 체크박스로 렌더링
- 이전엔 일반 불릿으로 표시되던 문제 수정

#### Enterprise MCP 설정 (`allowAllClaudeAiMcps`)
- `managed-mcp.json` 외에 **claude.ai 클라우드 MCP 커넥터 전체 자동 로드** 옵션
- 기업 관리 설정에서 활성화 가능

### 보안 수정 (중요)

| 항목 | 내용 |
|------|------|
| PowerShell 권한 우회 패치 | `cd..`, `cd\`, `cd~`, `X:` 등 내장 `cd` 함수가 작업 디렉터리를 감지 없이 변경하던 버그 수정 |
| Git worktree 샌드박스 수정 | 쓰기 허용 목록이 레포 루트 전체 대신 공유 `.git` 디렉터리만 포함하도록 수정 |
| PowerShell 접두사/와일드카드 규칙 | 네이티브 실행 파일·스크립트 사전 승인 처리 수정 |
| `PWD`/`OLDPWD`/`DIRSTACK` 추적 | 디렉터리 이동 명령 후 오래된 변수가 잔류하던 권한 분석 오류 수정 |

### 버그 수정
- `find` 명령이 대용량 디렉터리에서 macOS 파일/vnode 테이블 고갈 문제 수정
- `/ultraplan` 및 원격 세션에서 "Could not capture uncommitted changes" 오류 수정
- managed-settings 승인 다이얼로그 수락 후 터미널 프리즈 수정
- thinking 스피너 색상 오류 수정
- `/feedback` 리포트에 컨텍스트 압축 전 대화 내용 포함

## Claude Code v2.1.146 (2026-05-21)

> 출처: https://code.claude.com/docs/en/changelog | https://github.com/anthropics/claude-code/releases/tag/v2.1.146

### 신규 기능

- **`/code-review` effort 레벨**: `/simplify`에서 이름이 바뀐 명령에 선택적 effort 파라미터 추가
  - `/code-review high` — 높은 수준의 리뷰
  - `/code-review --comment` — GitHub PR에 인라인 댓글로 게시
- **Auto 모드 개선**: 사용자 또는 스킬이 명시적으로 필요로 할 때 `AskUserQuestion` 억제하지 않음

### 버그 수정

- **Windows PowerShell**: winget/Microsoft Store로 설치된 `pwsh`에서 "command line is invalid" 오류 수정 (v2.1.124 회귀)
- **MCP 페이지네이션**: `resources/list`, `resources/templates/list`, `prompts/list`에서 1페이지 이후 항목이 누락되던 문제 수정
- **백그라운드 세션**: 이미 승인된 권한을 재요청하던 문제 수정
- **CLAUDE_CODE_SUBAGENT_MODEL**: 자식 프로세스에 전달되지 않던 문제 수정
- **MCP `forceLoginOrgUUID`/`forceLoginMethod`** 정책 미적용 문제 수정
- Windows, GNOME Terminal, Agent SDK 등 다수의 UI 수정

## 사용 팁

### `/usage` 카테고리별 확인 (v2.1.149+)
- 한도를 가장 많이 쓰는 카테고리가 무엇인지 확인 후 최적화
- 특정 MCP 서버가 비용을 과다 소모하는 경우 교체·비활성화 검토

### 보안 패치 적용 권장 (v2.1.149)
- PowerShell 사용 환경이라면 v2.1.149+로 즉시 업데이트 권장
- Git worktree 샌드박스 범위 수정 포함

### `/code-review` 활용 (v2.1.146+)
```bash
/code-review          # 기본 리뷰
/code-review high     # 고강도 리뷰 (더 많은 이슈 탐지)
/code-review --comment  # GitHub PR 인라인 댓글로 게시
```
