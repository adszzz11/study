# Claude 업데이트 - 2026-05-14

## Claude Code v2.1.141 (2026-05-13)

> 출처: https://code.claude.com/docs/en/changelog

### 신규 기능 & 개선

- **훅 `terminalSequence` 필드**: 컨트롤 터미널 없이 데스크탑 알림·윈도우 타이틀·벨 출력 가능
- **`CLAUDE_CODE_PLUGIN_PREFER_HTTPS` 환경변수**: SSH 키 없이 HTTPS로 GitHub 플러그인 소스 클론
- **`ANTHROPIC_WORKSPACE_ID` 환경변수**: Workload Identity Federation 지원
- **`claude agents --cwd <path>`**: 특정 디렉토리로 세션 목록 범위 한정
- **`/feedback`에 최근 세션 포함**: 최근 24시간 또는 7일 세션 포함
- **Rewind 메뉴 "Summarize up to here"**: 이전 컨텍스트 압축 기능 추가
- **Auto mode 퍼미션 다이얼로그 개선**: `permissions.ask` 규칙으로 프롬프트 표시 이유 설명
- **파일 편집 퍼미션 프롬프트에 "IDE에서 diff 보기" 옵션 복원**
- **백그라운드 에이전트 퍼미션 모드 보존**: 현재 모드 유지
- **긴 thinking 시 스피너 피드백 개선**: 10초 후 amber 경고 표시
- **플러그인 메뉴 풀스크린 헤더 클릭 가능**

### 버그 수정 (30+개)

- Bedrock/Vertex/Foundry에서 소형 모델 미설정 시 백그라운드 side-query 수정
- Windows에서 잠긴 daemon pipe로 `claude daemon status`·`/doctor` 실패 수정
- 한 세션에서 `/model` 변경이 다른 세션의 autocompact 임계값을 바꾸던 문제 수정
- 셀 줄바꿈이 있는 마크다운 테이블이 세로 레이아웃으로 폴백되던 문제 수정
- 취소된 프롬프트가 위쪽 화살표 히스토리에서 제거되던 문제 수정
- Vim INSERT/VISUAL 모드에서 Ctrl+C 인터럽트가 작동하지 않던 문제 수정
- `enter`를 `chat:newline`으로 재바인딩 시 대체 `chat:submit` 키바인딩 동작 안 하던 문제 수정
- `AskUserQuestion` 팝업이 이전 채팅 내용을 가리던 문제 수정
- 세션 타이틀이 사용자 첫 프롬프트 대신 플러그인 알림에서 파생되던 문제 수정
- `/tui`가 백그라운드 셸·서브에이전트를 조용히 드롭하던 문제 수정
