## Claude Code v2.1.145 (2026-05-19)

> 출처: https://code.claude.com/docs/en/changelog

### 신규 기능

- **`claude agents --json`**: 실행 중인 Claude 세션 목록을 JSON으로 출력 → tmux-resurrect, 상태 바, 세션 피커 등 스크립팅에 활용 가능
- **OpenTelemetry 개선**: `claude_code.tool` OTEL span에 `agent_id`, `parent_agent_id` 속성 추가 및 백그라운드 서브에이전트 span 트레이스 부모 관계 수정
- **Status line JSON**: GitHub 레포 및 PR 정보 포함 (감지 시)
- **`/plugin` 탐색 개선**: 설치 전 플러그인의 commands, agents, skills, hooks, MCP/LSP 서버 목록 미리보기
- **터미널 UI**: `claude agents` 탭 타이틀에 입력 대기 세션 수 표시; 전체화면 모드에서 슬래시 명령·@-mention 제안 목록 마우스 호버·클릭 지원
- **훅 개선**: Stop·SubagentStop 훅 입력에 `background_tasks`, `session_crons` 필드 추가

### 보안 수정

- **권한 프롬프트 우회 버그 수정**: Bash 명령에서 허용 목록에 없는 환경변수에 대한 단순 변수 할당(`VAR=val`)이 자동 승인되던 문제 수정

### 버그 수정

- MCP 프롬프트 슬래시 명령이 서버 원시 오류를 표시하던 문제 수정
- 터미널 리사이즈·리포커스 후 스피너/경과시간 표시 멈춤 수정
- Windows PowerShell 5.1에서 교차 프로젝트 재개 힌트 실패 수정
- Agent View 응답 패널에서 음성 Push-to-Talk 미동작 수정
- 동시 생성 시 태스크 목록 무작위 순서 렌더링 수정
- `gh pr create` 직후 PR 배지 즉시 업데이트 수정
- 비ASCII 이름의 에이전트 팀 멤버 API 호출 실패 수정
- `/review`가 Classic Projects 레포에서 deprecated GraphQL 쿼리 사용하던 문제 수정
- Read 툴이 토큰 한도 초과 시 에러 대신 "PARTIAL view" 알림과 함께 첫 페이지 반환하도록 개선

## Claude Code v2.1.144 (2026-05-19)

> 출처: https://code.claude.com/docs/en/changelog
