# Conductor

> **한 줄 정의**: Claude Code 에이전트를 병렬로 실행하고 시각적으로 관리하는 네이티브 macOS 앱

## 개요

Melty Labs가 개발한 Conductor는 여러 Claude Code 세션을 동시에 실행하면서 각 세션을 독립된 git worktree에서 격리하는 GUI 기반 오케스트레이션 도구다. Y Combinator 지원을 받은 스타트업으로, "AI 에이전트의 IDE"를 표방한다.

---

## Quick Start

```bash
# 1. conductor.build에서 DMG 다운로드
# 2. 앱 설치 후 실행
# 3. 프로젝트 디렉토리 선택 → 에이전트 추가 → 병렬 작업 시작
```
