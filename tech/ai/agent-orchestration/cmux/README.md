# cmux

> **한 줄 정의**: AI 코딩 에이전트를 위해 설계된 네이티브 macOS 터미널 — 수직 탭, 분할 패널, 소켓 API 지원

## 개요

cmux(Claude Multiplexer)는 AI 코딩 에이전트를 동시에 여러 개 돌리기 위해 설계된 네이티브 macOS 터미널이다. Claude Code뿐 아니라 Gemini CLI, Codex, Aider, Kiro 등 모든 CLI 기반 에이전트를 지원한다. 수직 탭, 분할 패널, 읽지 않은 알림 배지 등 멀티 에이전트 워크플로우에 최적화된 UX를 제공한다.
