## 🛠️ **주요 업무 1: 소비잇 프로젝트**

### **핵심 기능**
- **🏦 국민카드 연동**: 채권 관리 및 매입심사 시스템
- **💳 할부 결제**: 할부 수납 및 청구 프로세스 관리
- **📊 정산 관리**: 가맹점 매출 및 정산 처리
- **🔄 배치 작업**: 파일 전송, 대사, 데이터 동기화

### **시스템 구조**
```
고객(하이브리드 앱) → Backend API → 국민카드 전문 통신
                          ↓
                    배치 작업 / 정산 시스템
```

### **주요 프로세스**
1. 신용평가 및 매입심사 요청 (→ 국민카드)
2. 결제 원장 등록 및 할부 적립금 등록
3. 가맹점 매출 처리 및 헥토 출금
4. 일일 배치를 통한 할부 수납 잔액 갱신

---

## 💰 **주요 업무 2: 환불/정산 시스템**

### **환불 프로세스**
```
환불 요청 → 계좌 유효성 검증(PAS) → 재무정보 시스템 탑재
     ↓              ↓                      ↓
  예정일 계산   계좌검증 완료         SAP 기표 처리
     ↓                                    ↓
환불 이체 실행 ← 환불완료/실패 처리 ← 결과 수신
```

### **핵심 포인트**
- ⏰ **환불 예정일 관리**: 영업일 기준 D+1 ~ D+2 처리
- 💸 **수수료 처리**: 거래금액, 환불액, 고객/다날 부담금 계산
- 🔍 **실패건 관리**: 환불 실패 케이스 모니터링 및 재처리
- 📋 **재무정보 연동**: SAP 시스템과의 정확한 기표 처리

### **기술적 고려사항**
- 계좌 검증 실패 시 후처리 방안
- 환불 예정일 경과 건에 대한 자동 실패 판단
- 재무정보 필드 정확도 관리
