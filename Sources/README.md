# Sources

원문, 논문, 공식 문서, 블로그, 저장소 링크를 정리하는 곳.

## Source Types

- Official docs
- Research papers
- Blog posts
- GitHub repositories
- Videos or talks
- Benchmarks and datasets

## Source Note Rules

- 원문 제목과 URL을 보존한다.
- 읽은 날짜와 접근 날짜를 분리한다.
- 요약은 내 말로 쓰고, 인용은 짧게 둔다.
- 관련 노트와 관련 MOC를 반드시 연결한다.

## Attachments

첨부 파일은 `Sources/attachments/`에 둔다.
