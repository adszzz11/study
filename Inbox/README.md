# Inbox

아직 분류하지 않은 질문, 링크, 캡처, 임시 메모를 두는 곳.

## Capture Checklist

- [ ] 원문 URL 또는 출처를 남겼는가?
- [ ] 왜 저장했는지 한 줄로 적었는가?
- [ ] 관련 MOC 후보를 적었는가?
- [ ] 처리 후 `tech/...`, [[Sources/README|Sources]], [[Projects/README|Projects]] 중 하나로 옮겼는가?

## Processing Rules

- 단순 링크는 [[Templates/Source Note|Source Note]]로 정리한다.
- 도구/프레임워크 학습은 [[Templates/Tool Study|Tool Study]]로 확장한다.
- 개념 노트는 [[Templates/Tech Note|Tech Note]]로 정리한다.
- 실행 가능한 실험은 [[Projects/README|Projects]]에 작업 단위로 옮긴다.
