# Konda 서비스 프로젝트 개요

## 1. 프로젝트 범위

#### 내부 개발 범위
| 영역 | 설명 |
|------|------|
| App 개발 | 모바일 애플리케이션 전체 개발 |
| BE 개발 (금융) | 금융 관련 백엔드 API 개발 |
| 배포 | 전체 서비스 배포 담당 |
| 최종 QA | 최종 품질 검수 |

#### 외주사 개발 범위
| 영역 | 설명 |
|------|------|
| 기획 | 서비스 기획 및 요구사항 정의 |
| 디자인 | UI/UX 디자인 |
| BE 개발 (비금융) | 비금융 관련 백엔드 API 개발 |
| 랜딩 페이지 | 서비스 소개 웹페이지 개발 |
| 1차 QA | 자체 품질 검수 |

---

## 2. 전체 아키텍처 개요

```
┌─────────────────────────────────────────────────────────────┐
│                        사용자                                │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                     Konda App (내부)                         │
│                    - iOS / Android                          │
└─────────────────────────────────────────────────────────────┘
                    │                    │
                    ▼                    ▼
┌───────────────────────────┐  ┌───────────────────────────┐
│     BE - 비금융 (외주사)     │  │     BE - 금융 (내부)        │
│                           │  │                           │
│  - 일반 서비스 API          │  │  - 금융 거래 API            │
│  - 사용자 관리 API          │  │  - 결제 API                │
│  - 콘텐츠 API              │  │  - 자산 관리 API            │
└───────────────────────────┘  └───────────────────────────┘
```

### 2.1 API 연동 구조
- **App ↔ BE(비금융)**: 직접 통신
- **App ↔ BE(금융)**: 직접 통신
- **BE(비금융) ↔ BE(금융)**: 직접 통신 없음 (App을 통해 필요시 연동)

### 2.2 저장소 구조
- **내부 Repository**: App, BE(금융) 코드
- **외부 Repository**: BE(비금융), 랜딩 페이지 코드
