## 설정 예시

### Kubernetes Deployment Rollback
```bash
# 롤백 실행
kubectl rollout undo deployment/myapp

# 특정 revision으로 롤백
kubectl rollout undo deployment/myapp --to-revision=2

# 롤아웃 이력 확인
kubectl rollout history deployment/myapp
```

### ArgoCD 자동 롤백
```yaml
apiVersion: argoproj.io/v1alpha1
kind: Application
metadata:
  name: myapp
spec:
  syncPolicy:
    automated:
      prune: true
      selfHeal: true
      allowEmpty: false
    retry:
      limit: 5
      backoff:
        duration: 5s
        factor: 2
        maxDuration: 3m
```

### Jenkins Pipeline 롤백
```groovy
pipeline {
    stages {
        stage('Deploy') {
            steps {
                script {
                    try {
                        // 배포 실행
                        deploy()
                        // Health Check
                        healthCheck()
                    } catch (Exception e) {
                        // 롤백 실행
                        rollback()
                        error("Deployment failed, rolled back")
                    }
                }
            }
        }
    }
}
```
