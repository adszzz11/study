# 실시간 거래 이관 API 개발 시 트랜잭션 일관성을 어떻게 보장했나요?

## 답변

실시간 거래 이관 API 개발 시, 여러 시스템(원장 시스템, 결제 시스템, 외부 VAN사) 간의 데이터 일관성을 보장하는 것이 가장 중요한 과제였습니다. 특히 금융 도메인에서는 단 한 건의 거래라도 누락되거나 중복되면 안 되기 때문에, 다음과 같은 전략으로 트랜잭션 일관성을 보장했습니다.

**핵심 전략:**

1. **Saga Pattern 적용**
   - 분산 환경에서 2PC는 성능 저하와 타임아웃 문제가 있어, Choreography 방식의 Saga Pattern 채택
   - 각 단계별 보상 트랜잭션(Compensating Transaction) 정의
   - 실패 시 자동 롤백 처리

2. **멱등성(Idempotency) 보장**
   - 모든 API 요청에 고유한 거래 ID 부여
   - 중복 요청 시 같은 결과 반환 (재시도 안전성)
   - Unique Index 활용한 DB 레벨 중복 방지

3. **이벤트 소싱 및 메시지 큐 활용**
   - Kafka를 통한 비동기 이벤트 처리
   - 모든 상태 변경을 이벤트로 기록
   - At-least-once 전송 보장 + 멱등성으로 정확히 한 번 처리

4. **분산 락(Distributed Lock)**
   - Redis를 활용한 분산 환경에서의 동시성 제어
   - 같은 거래에 대한 동시 처리 방지

5. **트랜잭션 로그 및 재시도 메커니즘**
   - 모든 거래 단계를 상태 머신으로 관리
   - 실패 시 자동 재시도 (Exponential Backoff)
   - 최종 실패 시 Dead Letter Queue로 수동 처리

## 핵심 키워드

- 트랜잭션 일관성
- 분산 트랜잭션
- ACID
- 보상 트랜잭션 (Saga Pattern)
- 멱등성 (Idempotency)

## 트랜잭션 일관성 전략

### 1. 2PC (Two-Phase Commit)

**적용하지 않은 이유:**
- Blocking Protocol: Coordinator가 응답하지 않으면 모든 참여자가 대기
- 타임아웃 시 일관성 보장 어려움
- 금융 시스템의 높은 트래픽 환경에서 성능 저하
- 외부 VAN사 시스템은 2PC를 지원하지 않음

**2PC가 적합한 경우:**
- 내부 DB 간의 트랜잭션 (같은 데이터센터)
- 거래량이 적고 일관성이 절대적으로 중요한 경우

### 2. Saga Pattern

**Choreography 방식 채택:**

```java
/**
 * 실시간 거래 이관 Saga
 *
 * 단계:
 * 1. 원장 시스템에서 잔액 차감
 * 2. 거래 이력 저장
 * 3. 외부 VAN사 승인 요청
 * 4. 정산 데이터 생성
 *
 * 실패 시 보상 트랜잭션:
 * - VAN 승인 실패 → 거래 이력 취소 + 잔액 복구
 * - 정산 데이터 생성 실패 → VAN 승인 취소 + 거래 이력 취소 + 잔액 복구
 */

@Service
public class PaymentSagaOrchestrator {

    @Autowired
    private LedgerService ledgerService;

    @Autowired
    private TransactionHistoryService transactionService;

    @Autowired
    private VanApprovalService vanService;

    @Autowired
    private SettlementService settlementService;

    @Autowired
    private KafkaTemplate<String, PaymentEvent> kafkaTemplate;

    /**
     * Saga 시작점
     */
    @Transactional
    public PaymentResult initiatePayment(PaymentRequest request) {
        String sagaId = generateSagaId();

        // Saga 상태 저장
        PaymentSaga saga = PaymentSaga.builder()
            .sagaId(sagaId)
            .status(SagaStatus.STARTED)
            .currentStep(SagaStep.DEDUCT_BALANCE)
            .request(request)
            .build();

        sagaRepository.save(saga);

        try {
            // Step 1: 원장 잔액 차감
            LedgerResult ledgerResult = ledgerService.deductBalance(
                request.getUserId(),
                request.getAmount(),
                sagaId
            );

            saga.setLedgerTransactionId(ledgerResult.getTransactionId());
            saga.setCurrentStep(SagaStep.SAVE_HISTORY);
            sagaRepository.save(saga);

            // Step 2: 거래 이력 저장
            TransactionHistory history = transactionService.saveHistory(
                ledgerResult,
                sagaId
            );

            saga.setHistoryId(history.getId());
            saga.setCurrentStep(SagaStep.VAN_APPROVAL);
            sagaRepository.save(saga);

            // Step 3: VAN 승인 요청 (비동기)
            publishVanApprovalEvent(saga);

            return PaymentResult.pending(sagaId);

        } catch (Exception e) {
            // 보상 트랜잭션 시작
            compensate(saga, e);
            throw new PaymentSagaException("결제 처리 실패", e);
        }
    }

    /**
     * VAN 승인 이벤트 발행
     */
    private void publishVanApprovalEvent(PaymentSaga saga) {
        PaymentEvent event = PaymentEvent.builder()
            .sagaId(saga.getSagaId())
            .eventType(EventType.VAN_APPROVAL_REQUESTED)
            .payload(saga.getRequest())
            .build();

        kafkaTemplate.send("payment-events", saga.getSagaId(), event);
    }

    /**
     * 보상 트랜잭션 (Rollback)
     */
    @Transactional
    private void compensate(PaymentSaga saga, Exception cause) {
        log.error("[Saga 보상 시작] sagaId={}, step={}, cause={}",
            saga.getSagaId(), saga.getCurrentStep(), cause.getMessage());

        saga.setStatus(SagaStatus.COMPENSATING);
        sagaRepository.save(saga);

        try {
            switch (saga.getCurrentStep()) {
                case VAN_APPROVAL:
                    // VAN 승인 실패 → 거래 이력 취소 + 잔액 복구
                    compensateVanApproval(saga);
                    // no break: cascade rollback

                case SAVE_HISTORY:
                    // 거래 이력 취소
                    compensateHistory(saga);
                    // no break: cascade rollback

                case DEDUCT_BALANCE:
                    // 잔액 복구
                    compensateBalance(saga);
                    break;
            }

            saga.setStatus(SagaStatus.COMPENSATED);
            saga.setCompensatedAt(LocalDateTime.now());
            sagaRepository.save(saga);

            log.info("[Saga 보상 완료] sagaId={}", saga.getSagaId());

        } catch (Exception e) {
            saga.setStatus(SagaStatus.COMPENSATION_FAILED);
            sagaRepository.save(saga);

            // Critical Alert - 수동 복구 필요
            alertService.sendCriticalAlert(
                "Saga 보상 실패",
                String.format("sagaId=%s, step=%s", saga.getSagaId(), saga.getCurrentStep())
            );

            log.error("[Saga 보상 실패] sagaId={}", saga.getSagaId(), e);
        }
    }

    private void compensateBalance(PaymentSaga saga) {
        if (saga.getLedgerTransactionId() != null) {
            ledgerService.refundBalance(
                saga.getRequest().getUserId(),
                saga.getRequest().getAmount(),
                saga.getLedgerTransactionId(),
                "Saga 보상"
            );
        }
    }

    private void compensateHistory(PaymentSaga saga) {
        if (saga.getHistoryId() != null) {
            transactionService.cancelHistory(saga.getHistoryId(), "Saga 보상");
        }
    }

    private void compensateVanApproval(PaymentSaga saga) {
        if (saga.getVanApprovalNo() != null) {
            vanService.cancelApproval(saga.getVanApprovalNo(), "Saga 보상");
        }
    }
}
```

```java
@Service
public class VanApprovalEventHandler {

    @KafkaListener(topics = "payment-events", groupId = "van-approval-group")
    public void handleVanApprovalEvent(PaymentEvent event) {
        if (event.getEventType() != EventType.VAN_APPROVAL_REQUESTED) {
            return;
        }

        PaymentSaga saga = sagaRepository.findBySagaId(event.getSagaId())
            .orElseThrow();

        try {
            // VAN 승인 요청
            VanApprovalResponse vanResponse = vanService.requestApproval(event.getPayload());

            saga.setVanApprovalNo(vanResponse.getApprovalNo());
            saga.setCurrentStep(SagaStep.SETTLEMENT);
            saga.setStatus(SagaStatus.VAN_APPROVED);
            sagaRepository.save(saga);

            // 다음 단계: 정산 데이터 생성
            publishSettlementEvent(saga);

        } catch (Exception e) {
            log.error("[VAN 승인 실패] sagaId={}", saga.getSagaId(), e);

            saga.setStatus(SagaStatus.VAN_APPROVAL_FAILED);
            sagaRepository.save(saga);

            // 보상 이벤트 발행
            publishCompensationEvent(saga);
        }
    }
}
```

## 구현 방법

### 멱등성 보장

```java
@Service
public class IdempotentPaymentService {

    @Autowired
    private RedisTemplate<String, String> redisTemplate;

    private static final String IDEMPOTENCY_KEY_PREFIX = "payment:idempotency:";
    private static final long IDEMPOTENCY_TTL = 24 * 60 * 60; // 24시간

    /**
     * 멱등성 보장 래퍼
     */
    @Transactional
    public PaymentResult processPayment(PaymentRequest request) {
        String idempotencyKey = request.getIdempotencyKey();

        if (idempotencyKey == null) {
            throw new IllegalArgumentException("idempotencyKey는 필수입니다.");
        }

        // 1. Redis에서 이전 결과 확인
        String cachedResult = redisTemplate.opsForValue()
            .get(IDEMPOTENCY_KEY_PREFIX + idempotencyKey);

        if (cachedResult != null) {
            log.info("[멱등성 처리] 이전 결과 반환: key={}", idempotencyKey);
            return objectMapper.readValue(cachedResult, PaymentResult.class);
        }

        // 2. DB에서 중복 확인 (Unique Index)
        Optional<Payment> existing = paymentRepository
            .findByIdempotencyKey(idempotencyKey);

        if (existing.isPresent()) {
            log.info("[멱등성 처리] DB에서 기존 거래 발견: key={}", idempotencyKey);
            PaymentResult result = PaymentResult.fromPayment(existing.get());
            cacheResult(idempotencyKey, result);
            return result;
        }

        // 3. 신규 거래 처리
        try {
            PaymentResult result = executePayment(request);

            // 4. 결과 캐싱
            cacheResult(idempotencyKey, result);

            return result;

        } catch (DuplicateKeyException e) {
            // Race Condition 발생 - 다시 조회
            log.warn("[멱등성 처리] Race condition 발생: key={}", idempotencyKey);
            Payment payment = paymentRepository.findByIdempotencyKey(idempotencyKey)
                .orElseThrow();
            PaymentResult result = PaymentResult.fromPayment(payment);
            cacheResult(idempotencyKey, result);
            return result;
        }
    }

    private void cacheResult(String idempotencyKey, PaymentResult result) {
        try {
            String json = objectMapper.writeValueAsString(result);
            redisTemplate.opsForValue().set(
                IDEMPOTENCY_KEY_PREFIX + idempotencyKey,
                json,
                IDEMPOTENCY_TTL,
                TimeUnit.SECONDS
            );
        } catch (Exception e) {
            log.error("[멱등성 캐싱 실패] key={}", idempotencyKey, e);
        }
    }
}

/**
 * DB 스키마: Unique Index로 중복 방지
 */
CREATE TABLE payment (
    payment_id BIGINT AUTO_INCREMENT PRIMARY KEY,
    idempotency_key VARCHAR(100) NOT NULL,
    user_id VARCHAR(20) NOT NULL,
    amount DECIMAL(15,2) NOT NULL,
    status VARCHAR(20) NOT NULL,
    created_at TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP,
    UNIQUE KEY uk_idempotency_key (idempotency_key)
);
```

```java
@Configuration
public class RetryConfiguration {

    /**
     * Exponential Backoff 재시도 정책
     */
    @Bean
    public RetryTemplate paymentRetryTemplate() {
        RetryTemplate retryTemplate = new RetryTemplate();

        // 재시도 정책: 최대 3회
        SimpleRetryPolicy retryPolicy = new SimpleRetryPolicy();
        retryPolicy.setMaxAttempts(3);
        retryTemplate.setRetryPolicy(retryPolicy);

        // Exponential Backoff: 1초, 2초, 4초
        ExponentialBackOffPolicy backOffPolicy = new ExponentialBackOffPolicy();
        backOffPolicy.setInitialInterval(1000L);
        backOffPolicy.setMultiplier(2.0);
        backOffPolicy.setMaxInterval(10000L);
        retryTemplate.setBackOffPolicy(backOffPolicy);

        return retryTemplate;
    }
}

@Service
public class PaymentRetryService {

    @Autowired
    private RetryTemplate paymentRetryTemplate;

    /**
     * 재시도 가능한 결제 처리
     */
    public PaymentResult processWithRetry(PaymentRequest request) {
        return paymentRetryTemplate.execute(context -> {
            log.info("[재시도] 시도 횟수: {}", context.getRetryCount() + 1);

            try {
                return paymentService.processPayment(request);

            } catch (VanTimeoutException e) {
                // 재시도 가능한 예외
                log.warn("[재시도 대상] VAN 타임아웃", e);
                throw e;

            } catch (InsufficientBalanceException e) {
                // 재시도 불가능한 예외
                log.error("[재시도 불가] 잔액 부족", e);
                throw new TerminatedRetryException("재시도 불가능한 오류", e);
            }

        }, context -> {
            // 복구 콜백 (모든 재시도 실패 시)
            log.error("[재시도 최종 실패] 횟수: {}", context.getRetryCount());

            // Dead Letter Queue로 이동
            sendToDeadLetterQueue(request);

            return PaymentResult.retryExhausted();
        });
    }

    private void sendToDeadLetterQueue(PaymentRequest request) {
        kafkaTemplate.send("payment-dlq", request);
    }
}
```

### 1. 부분 실패 (VAN 승인 후 정산 실패)

```java
/**
 * 시나리오: VAN 승인은 성공했으나 정산 데이터 생성 실패
 * 대응: VAN 승인 취소 API 호출
 */
@Service
public class SettlementFailureHandler {

    @Transactional
    public void handleSettlementFailure(PaymentSaga saga) {
        log.error("[정산 실패] sagaId={}, vanApprovalNo={}",
            saga.getSagaId(), saga.getVanApprovalNo());

        try {
            // VAN 승인 취소 요청
            VanCancelRequest cancelRequest = VanCancelRequest.builder()
                .approvalNo(saga.getVanApprovalNo())
                .amount(saga.getRequest().getAmount())
                .reason("정산 데이터 생성 실패")
                .build();

            VanCancelResponse cancelResponse = vanService.cancelApproval(cancelRequest);

            if (cancelResponse.isSuccess()) {
                log.info("[VAN 승인 취소 성공] approvalNo={}", saga.getVanApprovalNo());

                // 거래 이력 취소
                transactionService.cancelHistory(saga.getHistoryId(), "정산 실패로 인한 취소");

                // 잔액 복구
                ledgerService.refundBalance(
                    saga.getRequest().getUserId(),
                    saga.getRequest().getAmount(),
                    saga.getLedgerTransactionId(),
                    "정산 실패 보상"
                );

                saga.setStatus(SagaStatus.COMPENSATED);
                sagaRepository.save(saga);

            } else {
                throw new VanCancelException("VAN 취소 실패: " + cancelResponse.getMessage());
            }

        } catch (Exception e) {
            // VAN 취소 실패 - Critical Alert
            saga.setStatus(SagaStatus.COMPENSATION_FAILED);
            sagaRepository.save(saga);

            alertService.sendCriticalAlert(
                "VAN 승인 취소 실패 - 수동 처리 필요",
                String.format("sagaId=%s, approvalNo=%s", saga.getSagaId(), saga.getVanApprovalNo())
            );

            log.error("[VAN 취소 실패] sagaId={}", saga.getSagaId(), e);
        }
    }
}
```

```java
/**
 * 시나리오: VAN 요청 후 타임아웃 발생, 실제 승인 여부 불명
 * 대응: 승인 조회 API로 상태 확인
 */
@Service
public class NetworkFailureHandler {

    @Transactional
    public void handleTimeout(PaymentSaga saga) {
        log.warn("[타임아웃 발생] sagaId={}", saga.getSagaId());

        try {
            // 5초 대기 후 승인 조회
            Thread.sleep(5000);

            VanInquiryRequest inquiryRequest = VanInquiryRequest.builder()
                .merchantId(saga.getRequest().getMerchantId())
                .transactionId(saga.getSagaId())
                .build();

            VanInquiryResponse inquiryResponse = vanService.inquiryApproval(inquiryRequest);

            if (inquiryResponse.isApproved()) {
                // 실제로 승인되어 있음
                log.info("[타임아웃 후 승인 확인] sagaId={}", saga.getSagaId());

                saga.setVanApprovalNo(inquiryResponse.getApprovalNo());
                saga.setStatus(SagaStatus.VAN_APPROVED);
                sagaRepository.save(saga);

                // 정산 진행
                publishSettlementEvent(saga);

            } else {
                // 승인 안됨 - 보상 트랜잭션
                log.info("[타임아웃 후 미승인 확인] sagaId={}", saga.getSagaId());
                compensate(saga, new VanApprovalException("VAN 승인 실패"));
            }

        } catch (Exception e) {
            // 조회도 실패 - 상태 불명
            saga.setStatus(SagaStatus.UNKNOWN);
            sagaRepository.save(saga);

            alertService.sendCriticalAlert(
                "거래 상태 불명 - 수동 확인 필요",
                String.format("sagaId=%s", saga.getSagaId())
            );

            log.error("[조회 실패] sagaId={}", saga.getSagaId(), e);
        }
    }
}
```

### 3. 동시성 충돌 (분산 락)

```java
@Service
public class DistributedLockService {

    @Autowired
    private RedissonClient redissonClient;

    /**
     * 분산 락으로 동시 처리 방지
     */
    public <T> T executeWithLock(String lockKey, long waitTime, long leaseTime,
                                   Supplier<T> supplier) {
        RLock lock = redissonClient.getLock(lockKey);

        try {
            boolean acquired = lock.tryLock(waitTime, leaseTime, TimeUnit.SECONDS);

            if (!acquired) {
                throw new LockAcquisitionException("락 획득 실패: " + lockKey);
            }

            return supplier.get();

        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
            throw new LockAcquisitionException("락 획득 중 인터럽트", e);

        } finally {
            if (lock.isHeldByCurrentThread()) {
                lock.unlock();
            }
        }
    }
}

// 사용 예시
public PaymentResult processPaymentWithLock(PaymentRequest request) {
    String lockKey = "payment:lock:" + request.getUserId();

    return lockService.executeWithLock(lockKey, 3, 10, () -> {
        return paymentService.processPayment(request);
    });
}
```

```java
@Component
public class SagaMonitoringService {

    @Scheduled(fixedDelay = 60000)  // 1분마다 실행
    public void monitorStuckSagas() {
        LocalDateTime threshold = LocalDateTime.now().minusMinutes(10);

        // 10분 이상 진행 중인 Saga 조회
        List<PaymentSaga> stuckSagas = sagaRepository
            .findByStatusInAndCreatedAtBefore(
                Arrays.asList(SagaStatus.STARTED, SagaStatus.VAN_APPROVED),
                threshold
            );

        if (!stuckSagas.isEmpty()) {
            log.warn("[Stuck Saga 감지] 개수: {}", stuckSagas.size());

            for (PaymentSaga saga : stuckSagas) {
                // 재시도 시도
                retrySaga(saga);
            }
        }
    }

    @Scheduled(fixedDelay = 300000)  // 5분마다 실행
    public void monitorCompensationFailures() {
        List<PaymentSaga> failedCompensations = sagaRepository
            .findByStatus(SagaStatus.COMPENSATION_FAILED);

        if (!failedCompensations.isEmpty()) {
            alertService.sendCriticalAlert(
                "보상 트랜잭션 실패 - 수동 처리 필요",
                String.format("실패 건수: %d", failedCompensations.size())
            );
        }
    }

    /**
     * Saga 메트릭 수집
     */
    @Scheduled(fixedDelay = 60000)
    public void collectMetrics() {
        long totalSagas = sagaRepository.count();
        long successSagas = sagaRepository.countByStatus(SagaStatus.COMPLETED);
        long failedSagas = sagaRepository.countByStatus(SagaStatus.COMPENSATED);
        long unknownSagas = sagaRepository.countByStatus(SagaStatus.UNKNOWN);

        meterRegistry.gauge("saga.total", totalSagas);
        meterRegistry.gauge("saga.success", successSagas);
        meterRegistry.gauge("saga.failed", failedSagas);
        meterRegistry.gauge("saga.unknown", unknownSagas);

        double successRate = (double) successSagas / totalSagas * 100;
        meterRegistry.gauge("saga.success.rate", successRate);
    }
}
```
