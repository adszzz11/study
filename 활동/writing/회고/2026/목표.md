# 2026년 목표

> [!abstract] 2025년 성과를 바탕으로 설정한 2026년 목표
> 테마: **"다시, 나"의 해**

---

## 2025년 → 2026년 비교

| 영역 | 2025년 실적 | 2026년 목표 | 변화 |
|:-----|:------------|:------------|:----:|
| 체중 | 120kg | 100kg 미만 | **-20kg** |
| 운동 | 초반만 | 주 2회 이상 | ↑ |
| 발표/연사 | 3회 | 4회 이상 | +1 |
| 블로그 | 4편 | 12편 | **x3** |
| 독서 | 3권 | 6권 | **x2** |
| 강의 | 30개+ 미수강 | AI, Backend 우선 완료 | 우선순위 |
| 새 인연 | 30명 | 유지 | = |
| 번아웃 지수 | 4/10 | 4 이하 유지 | = |

---

## Keep / Stop / Start

### Keep (유지)

> [!success] 2025년에 잘 했던 것, 계속할 것

- 커뮤니티 활동과 연사 경험
- 새로운 기술 탐색하는 습관
- 사람들과 소통하며 배우는 자세

### Stop (중단)

> [!warning] 2025년에 문제였던 것, 그만할 것

- "바쁘니까" 핑계로 건강/생활 미루기
- 강의 구매만 하고 안 보기
- 미룬 일에 대한 죄책감만 갖고 행동 안 하기

### Start (시작)

> [!tip] 2026년에 새로 시작할 것

- 주간 리뷰 습관 (일요일 저녁 10분)
- 운동 루틴 (주 2회, 헬스/홈트)
- 이직 준비 체계적으로
- 1주 내 처리 원칙

## 세부 지표

### 1. 건강

> [!danger] 2025년 가장 큰 후회. 최우선 과제.

| 지표 | 현재 | 목표 | 마일스톤 | 측정 주기 |
|:-----|:-----|:-----|:---------|:----------|
| 체중 | 120kg | <100kg | Q1: 115kg / Q2: 110kg / Q3: 105kg / Q4: <100kg | 주 1회 |
| 운동 횟수 | 0회/주 | 2회/주 | 꾸준히 유지 | 주간 |
| 운동 종류 | - | 헬스, 홈트 | 어떤 운동이든 OK | - |

**실행 계획:**
- [ ] 체중계 구매 (없으면)
- [ ] 매주 같은 요일/시간에 측정
- [ ] 운동 기록 앱 또는 노트 활용
- [ ] 분기마다 5kg 감량 페이스

---

### 2. 커뮤니티 / 영향력

> [!success] 2025년 가장 큰 성장 동력. 확장한다.

| 지표 | 2025년 | 2026년 목표 | 분기별 목표 | 측정 주기 |
|:-----|:-------|:------------|:-----------|:----------|
| 발표/연사 | 3회 | 4회 이상 | Q1: 1 / Q2: 1 / Q3: 1 / Q4: 1 | 분기 |
| 블로그 포스트 | 4편 | 12편 | 월 1편 | 월간 |
| 팟캐스트/인터뷰 | ? | 2회 이상 | - | 연간 |
| 새로운 인연 | 30명 | 유지 또는 확장 | - | 연간 |

**실행 계획:**
- [ ] 발표할 컨퍼런스/밋업 리스트 작성 (Q1)
- [ ] 블로그 주제 백로그 10개 이상 만들기
- [ ] 매월 첫째 주에 해당 월 포스트 주제 확정
- [ ] 커뮤니티 행사 월 1회 이상 참석

### 3. 학습

> [!warning] 30개+ 미수강. 전부는 무리. 우선순위 집중.

| 지표 | 현재 | 2026년 목표 | 분기별 목표 | 측정 주기 |
|:-----|:-----|:------------|:-----------|:----------|
| 강의 - AI | 미수강 | 완료 | Q1-Q2 집중 | 분기 |
| 강의 - Backend Arch | 미수강 | 완료 | Q2-Q3 집중 | 분기 |
| 독서 | 3권/년 | 6권/년 | Q1: 1-2권 / Q2: 1-2권 / Q3: 1-2권 / Q4: 1-2권 | 분기 |
| 새 강의 구매 | 충동구매 | 기존 완료 후에만 | - | - |

**우선순위 강의 목록:**
1. AI 관련: (강의명 작성 필요)
2. Backend Architecture: (강의명 작성 필요)

**실행 계획:**
- [ ] 미수강 강의 전체 목록 정리
- [ ] 상위 5개 우선순위 선정
- [ ] 주 2-3시간 학습 시간 확보
- [ ] 새 강의 구매 전 "언제 볼 것인지" 캘린더 등록 필수

### 4. 이직

> [!note] 역량을 마음껏 펼칠 수 있는 곳으로

| 지표 | 현재 | 목표 | 마감 | 상태 |
|:-----|:-----|:-----|:-----|:----:|
| 이력서 업데이트 | 미완료 | 완료 | Q1 | ⬜ |
| 포트폴리오 정리 | 미완료 | 완료 | Q1 | ⬜ |
| 목표 기업 리스트 | 없음 | 10곳 | Q1 | ⬜ |
| 기술 면접 준비 | - | 완료 | Q1-Q2 | ⬜ |
| 이직 완료 | - | Y | 상반기 | ⬜ |

**원하는 역할:**
- 백엔드 개발자
- 데브옵스 엔지니어
- 테크리드
- 플랫폼 엔지니어
- (열려있음)

**원하는 환경:**
- 좋은 일, 재미있는 일
- 내가 도움이 될 수 있는 곳
- 역량을 마음껏 펼칠 수 있는 곳

**실행 계획:**
- [ ] 이력서 초안 작성
- [ ] 제품 출시 경험 스토리 구조화
- [ ] LGTM 프로젝트 경험 정리 (무산됐지만 배운 점)
- [ ] 목표 기업 10곳 리서치
- [ ] 기술 면접 예상 질문 정리

### 5. 생활 / 습관

> [!info] 2025년에 없었던 것. 기반부터 만든다.

| 지표 | 현재 | 목표 | 측정 주기 |
|:-----|:-----|:-----|:----------|
| 주간 리뷰 | 없었음 | 일요일 저녁 10분 | 주간 |
| 청소 | 불규칙 | 주 1회 (일요일) | 주간 |
| 미루는 일 처리 | 높음 | 1주 내 처리 | 주간 |
| 집안일 | 미룸 | 즉시 또는 당일 | 일간 |

**주간 리뷰 체크리스트:**
- [ ] 이번 주 완료한 것
- [ ] 다음 주 집중할 것
- [ ] 미룬 일 있는지 점검
- [ ] 운동 횟수 체크
- [ ] 체중 기록

**실행 계획:**
- [ ] 일요일 저녁 알람 설정
- [ ] 주간 리뷰 템플릿 만들기
- [ ] 청소 = 일요일 저녁 고정

### 6. 마인드

> [!note] 현재 4/10까지 회복. 유지가 목표.

| 지표 | 현재 | 목표 | 측정 주기 |
|:-----|:-----|:-----|:----------|
| 번아웃 지수 | 4/10 | ≤4 유지 | 월 1회 |
| 회복 시간 | ? | 주 2시간 | 주간 |
| 감정 기록 | 없음 | 힘들 때 글로 정리 | 수시 |

**에너지 채우는 것:**
- 운동
- 정리된 공간
- 의미 있는 대화
- 하고 싶은 거 하기 (게임, 산책, 음악, 넷플릭스 등)

**에너지 빼앗는 것:**
- 미룬 일의 죄책감
- 어지러운 환경
- 의미 없는 야근

**실행 계획:**
- [ ] 월말에 번아웃 지수 자가 점검
- [ ] 주 2시간 "나만의 시간" 캘린더에 블록
- [ ] 힘들 때 바로 글로 정리하는 습관
